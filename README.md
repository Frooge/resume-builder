# Resume Builder

A simple web app for building and exporting resumes. Edit sections live, tweak appearance, and download as PDF or DOCX.

**Live:** [frooge.github.io/resume-builder](https://frooge.github.io/resume-builder/)

## Run locally

```bash
npm install
npm run dev
```

## Contribute

Contributions are welcome. Fork the repo, make your changes, and open a pull request.

Before submitting:

```bash
npm run lint
npm run build
```
